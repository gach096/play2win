# Play2Win - Plataforma estilo Steam

Proyecto académico desarrollado para la asignatura Laboratorio IV de la **Tecnicatura Universitaria en Programación** (Plan 2003) - UTN Facultad Regional Haedo.

Realizado en colaboración con **Federico Ariel Gomez**.

---

## Descripción del Proyecto

Play2Win es una **plataforma web tipo Steam** que permite:
- Navegar un catálogo de juegos
- Sistema de usuarios (registro e inicio de sesión)
- Carrito de compras
- Panel de administración básico
- Gestión de productos/juegos

Es una aplicación **full-stack** con frontend atractivo y backend funcional.

---

## Tecnologías utilizadas

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP
- **Base de datos**: MySQL / SQL
- **Otros**: Diseño responsive

---

## Servicios utilizados

- **XAMPP**: Testeo de funcionamiento y programación de base de datos.
- **Sublimetext**: Programación general de la página.
- **InfinityFree**: Servicio de host gratuito.
- **FileZilla**: Gestión de archivos del servidor.
